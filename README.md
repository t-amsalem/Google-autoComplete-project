# google-project-t-amsalem

פרויקט זה תומך ב2 פונקציות עיקריות:
פונקציית אתחול - הפונקציה מקבלת רשימה של מקורות טקסט שעל בסיסם ירוץ מנוע החיפוש, כל מקור מכיל אוסף של משפטים. 
פונקציית השלמה - הפונקציה מקבלת מחרוזת - אשר מהווה את הטקסט שהמשתמש הקליד, הפונקציה מחזירה את חמש ההשלמות הטובות ביותר.
המטרה של פעולת ההשלמה היא להקל על המשתמש למצוא את המשפט המתאים ביותר.

נגדיר התאמה בין פסוק לבין טקסט שהמשתמש הקליד אם:
? הטקסט הוא תת מחרוזת של הפסוק (זה כולל תחילת, אמצע או סוף הפסוק).
? טקסט שבו אם נבצע תיקון אחד אז הטקסט יהווה תת מחרוזת של הפסוק.  

לכל משפט יש ציון, במידה ויש צורך לבצע שינוי בטקסט כדי שיתאים לפסוק המקורי מורידים בציון.
